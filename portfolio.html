Microsoft Copilot Mobile

Multi-agent development system

I created and shipped a production multi-agent development system for Microsoft Copilot Mobile. It gives PMs, designers, and engineers a safer path from product intent to mobile changes, with evals, review, and recovery thinking in the loop.

  • Role Original builder and active developer
  • Signal PM and Design teammates used the system directly to create and merge a meaningful volume of production PRs with engineering review, enough that review capacity became the next bottleneck
  • Focus Agent workflows, eval gates, risk-aware review, and quality visibility

Atlas

Private AI workflow and eval system

Atlas is my private system for studying how agents behave around real work: tool calls, context, local commands, sensitive data, scheduled jobs, and eval replay. The public page shows the shape of the system without exposing private records or implementation details.

  • Role Creator and primary user
  • Shape CLI spine, agent layer, private data boundary, always-on jobs, and eval harness
  • Learning Reliable agents need source-of-truth discipline and recovery paths, not just better prompts
